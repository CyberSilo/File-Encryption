# File-Encryption
<h1>VeraCrypt - File Encryption</h1>

<h2>Description</h2>
Project consists of a simple encrypted file container using a windows application 
<br />


<h2>Utilities Used</h2>

- <b>VeraCrypt</b>
- <b>File Explorer</b>

<h2>Environments Used </h2>

- <b>Windows 10 Pro</b> (22H2)
